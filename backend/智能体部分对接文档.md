### Psychological_Counseling
- **作用**
利用 Coze 设定特定的 Agent 角色和 Prompt（提示词），提供具备共情能力的对话体验 。
- **输出结果事例**

字段名	数据类型	示例数据	备注说明
id	Integer	1	主键，自动递增
real_student_id	String	"20230101"	学校真实学号（供老师在管理后台检索筛选）
fake_student_id	String	"STUDENT_HASH_7a9b"	传给 Coze 的假学号（用于审计对账和云端标识）
warning_level	String	"黄色预警"	对应的预警级别
survey_indicator	String	"焦虑情绪、动力不足"	调查问卷暴露的各项指标变化
