## 1. 登录页

### 页面名
`LoginPage`

### 页面功能
- **账号密码登录**：用户输入用户名和密码，调用 `POST /api/auth/login`，登录成功后将 JWT 存入 localStorage（记住我）或 sessionStorage。
- **一网通 SSO 登录入口**：提供"其他入口"区域，可跳转一网通 SSO 登录流程，回调调用 `POST /api/auth/sso/callback` 换取系统 JWT。
- **获取用户信息**：登录后调用 `GET /api/auth/info` 获取当前用户信息（姓名、角色、手机号等），并根据角色（学生/教师/管理员）跳转至对应页面。
- **记住我**：勾选后用户名和 token 持久化到 localStorage，下次打开页面自动填充用户名。
- **密码显隐切换**：点击眼睛图标切换密码输入框的明文/密文模式。
- **输入校验与错误提示**：前端校验用户名和密码非空；接口异常时展示错误信息。

## 2. 首页

### 页面名
`Home`

### 页面功能
- **导航栏**：顶部展示学校 Logo（nui_small_white.png），提供"考务资料"、"教学帮扶"、"心理指导"三个下拉菜单入口。

## 3. 对话页（智能问答）

### 页面名
`Dialogue`

### 页面功能
- **消息列表**：展示用户和 AI 助手的对话记录，用户消息蓝色气泡靠右，AI 回复白色气泡靠左，支持换行文本渲染。
- **自动发送首条消息**：接收首页传递的问题和模块类型参数，进入页面后自动调用 `POST /api/chat/text` 发送首条消息。
- **文本消息发送**：用户在底部输入框输入文字，Enter 发送、Shift+Enter 换行，调用 `POST /api/chat/text`，传入 query 和 moduleType。
- **流式回复（预留）**：可对接 `POST /api/chat/ai-core/stream` 实现 AI 回复逐字打印效果。
- **AI 思维链展示（预留）**：可对接 `POST /api/chat/ai-core` 展示问题改写、分类、命中状态、引用来源等调试信息。
- **打字指示器**：AI 回复生成过程中展示三点跳动动画。
- **心理倾诉（预留）**：可对接 `POST /api/chat/psychological`，学生输入心理倾诉内容获取心理咨询回复。
- **语音输入（预留）**：可对接 `POST /api/chat/voice`，上传录音文件进行 ASR 识别并获取 TTS 语音回复。
- **转人工客服（预留）**：可对接 `POST /api/chat/transfer-to-human`，用户申请人工客服介入。
- **点踩上报（预留）**：用户对不满意的 AI 回答可调用 `POST /api/chat/report-unrecognized` 上报未识别问题。
- **返回首页**：点击"返回首页"按钮回到首页。

## 4. 知识库管理页（管理员）

### 页面名
`KnowledgeBase`（待实现）

### 页面功能（基于 API 设计）
- **文档上传**：管理员上传 Word/Excel/PDF/TXT 文档，调用 `POST /api/kb/upload`，支持选择所属模块（考务通知/教学运行/学业帮扶/心理辅导）。
- **文档解析记录列表**：展示上传文档的解析状态（待解析/解析中/成功/失败），调用 `GET /api/kb/documents`，支持按状态过滤。
- **问答词条列表**：分页或全量展示已落库的问答词条，调用 `GET /api/kb/entries`，支持按关键字、模块、状态、来源搜索过滤。
- **手动新增词条**：逐条添加问答，调用 `POST /api/kb/entries`，填写问题、答案、模块、来源类型。
- **编辑词条**：修改现有词条的问题、答案、状态、模块等字段，调用 `PUT /api/kb/entries`。
- **禁用/删除词条**：调用 `DELETE /api/kb/entries/{id}` 移除词条及向量索引。

## 6. 数据统计页（管理员）

### 页面名
`Dashboard / DataStat`（待实现）

### 页面功能（基于 API 设计）
- **常见问题词云/榜单**：调用 `GET /api/stat/hot-questions`，展示近 N 天热度最高的问题，可接入 ECharts 词云或排行榜。
- **兜底闭环概览**：调用 `GET /api/stat/fallback-overview`，展示命中分布（未命中/弱命中/强命中数量）、未命中待处理量、Top 未识别问题列表。
- **未识别问题管理**：调用 `GET /api/admin/unrecognized/list` 分页查看未识别问题，支持按状态过滤；调用 `POST /api/admin/unrecognized/update-status` 标记已处理。
- **手动重建统计**：调用 `POST /api/stat/hot-questions/rebuild`，指定日期或天数重建常见问题统计。

## 9. 微信公众号 H5 页

### 页面名
`WechatH5`（待实现）

### 页面功能（基于 API 设计）
- **微信免密登录**：用户通过微信授权获取 code，调用 `GET /api/wechat/login?code=xxx` 换取系统 JWT，进入 H5 问答界面。
- **智能问答（同对话页）**：复用对话页逻辑，调用 `POST /api/chat/text` 进行文本问答。
- **语音提问**：通过微信 JS-SDK 录音，上传至 `POST /api/chat/voice` 获取语音识别和 TTS 回复。
- **常见问题推荐**：调用 `GET /api/chat/suggested-questions` 展示推荐问题气泡。
- **转人工客服**：调用 `POST /api/chat/transfer-to-human` 申请人工介入。
- **WebSocket 推送**：建立 `ws://host/ws/chat/user/{userId}` 连接，接收客服回复和系统通知。
