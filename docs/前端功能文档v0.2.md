# 智能问答辅助系统 — 前端功能文档 v0.2

> **变更说明**: v0.2 在 v0.1（基于 `backend/API接口文档.md`）基础上，结合 `Coze智能助手任务分配表.xlsx` 补充了角色细分、新增功能模块（公众号集成、主动推送、成长档案、个性化资源、数据脱敏等）以及 WebSocket 实时通讯等关键技术细节。

---

## 1. 角色定义

| 角色 | 标识 | 说明 |
|------|------|------|
| 学生 | `student` | 核心服务对象，通过 AI 对话获取学业、心理、校园生活等方面的智能问答服务 |
| 客服 | `support` / `staff` | 人工客服坐席，通过 WebSocket 实时接管 AI 转人工的会话，与用户一对一文字沟通 |
| 辅导员 | `counselor` | 面向所带班级学生，查看学业预警、生成帮扶策略、生成长档案与帮扶成效报告 |
| 教务老师 | `teacher` / `academic` | 管理知识库（上传/编辑），查看高频热点统计，处理未识别问题 |
| 系统管理员 | `admin` | 最高权限，涵盖上述全部管理功能，并可管理用户权限与系统配置 |

> **说明**: 后端 `getUserInfo` 返回的 `role` 字段值需与后端最终确认对齐。前端按"学生端"和"管理端"两大视角划分，管理端内部按角色做功能级权限控制（见第 6 节）。

## 3. 通用功能（所有角色）

### 3.1 登录认证

| 功能点 | 对应 API | 说明 |
|--------|----------|------|
| 账号密码登录 | `POST /api/auth/login` | 输入用户名+密码，获取 JWT Token |
| 一网通 SSO 登录 | `POST /api/auth/sso/callback` | 携带 SSO 凭证换取系统 Token |
| 微信 OAuth 登录 | `GET /api/wechat/login` | 微信授权回调，传入 code 换取系统 Token |
| 获取用户信息 | `GET /api/auth/info` | 登录后获取当前用户角色、用户名等信息 |
| 登出 | `POST /api/auth/logout` | 退出登录，注销 Token |

**页面内容**：
- 登录表单（用户名 + 密码）
- "一网通登录" 按钮 → 跳转学校统一认证页面
- "微信登录" 入口 → 展示公众号二维码
- 登录失败提示、Token 过期自动跳转登录页
- 根据返回的 `role` 字段自动路由到学生端或管理端

### 3.2 全局设计规范

| 要点 | 说明 | 任务来源 |
|------|------|----------|
| 专属 IP 形象 | 系统定制专属吉祥物/虚拟形象，统一出现在对话页、加载态、欢迎页 | 任务 1 |
| 多端响应式适配 | 学生端需同时适配 PC Web、手机浏览器、微信公众号内嵌页面三端 | 任务 1、2 |
| 情感化交互 | 加载时显示形象动画、问候语个性化（分时段）、空状态趣味提示 | 任务 1 |
| 数据脱敏展示 | 前端严格按权限展示敏感字段，管理端学生列表中的姓名、学号等做部分掩码处理 | 任务 15 |

## 4. 学生端功能

### 4.1 AI 智能问答对话（学生端核心）

**对应 API**：

| 接口 | 路径 |
|------|------|
| 发送文本消息 | `POST /api/chat/text` |
| 发送语音消息 | `POST /api/chat/voice` |
| 获取推荐问题 | `GET /api/chat/suggested-questions` |
| 申请转人工 | `POST /api/chat/transfer-to-human` |
| 上报未识别回答 | `POST /api/chat/report-unrecognized` |
| 获取座机推送 | 待后端补充 `GET /api/business/phone` |

**页面内容**：

- **顶部栏**: 专属 IP 形象 + 系统名称 + 当前在线状态指示
- **对话区**:
  - 聊天气泡形式展示提问与 AI 回答
  - 支持滚动加载历史记录
  - AI 回答中显示"正在输入..."动画（专属形象动效）
  - 答案底部提供"有帮助/无帮助"评价按钮
- **推荐问题区**（欢迎态 / 对话为空时）:
  - "猜你想问" 气泡列表 → 来自 `GET /api/chat/suggested-questions`
  - 点击即可快速提问
- **业务关联推送卡片**（对话中动态出现）:
  - 当 AI 识别到用户问题涉及特定业务（如选课、重修、考务），自动在回答下方推送关联的教务老师座机号码 → 对应任务 6
  - 卡片样式：图标 + 业务名称 + 联系电话
- **输入区**（底部固定）:
  - 文本输入框 + 发送按钮
  - 麦克风按钮 → 录音 → 调用 `POST /api/chat/voice`
  - 录音时显示波形动画，松手发送
- **转人工按钮**: 始终可见，点击后调 2.4
  - 提交后展示排队状态："您排在第 X 位，预计等待 X 分钟"
  - 人工接入时切换为 WebSocket 实时对话模式（见 4.1.1）
- **点踩/反馈按钮**: 当 AI 返回兜底话术时，显示"反馈此问题"按钮 → 调 2.8 上报

#### 4.1.1 人工转接 WebSocket 实时对话

> 对应任务 5：未识别问题转人工闭环

| 要点 | 说明 |
|------|------|
| 技术方案 | 前端建立 WebSocket 连接，与后端实时通讯服务器通信 |
| 状态切换 | AI 对话 → 排队等待 → 人工接入 → 客服回复 → 会话结束返回 AI |
| 前端状态展示 | 等待中（排队序号）、接待中（客服头像+姓名）、已结束 |
| 消息同步 | WebSocket 双向收发文本消息，前端即时渲染到对话区 |
| 断线重连 | WebSocket 断线时自动重连，并提示用户"连接断开，正在重连..." |

### 4.2 心理倾诉

**对应 API**：

| 接口 | 路径 |
|------|------|
| 心理咨询工作流 | `POST /api/chat/psychological` |

> 对应任务 13：Coze Agent + 学生画像 → 调试倾诉 Agent 的共情 Prompt

**页面内容**：
- 温暖、柔和的 UI 色调（区别于常规问答页的冷色调）
- 对话形式输入学生情绪描述、困扰等
- AI 返回安抚指导 / 共情话术（由 Coze 心理倾诉 Agent 生成）
- 情绪状态标记：若后端返回 `stress_level` 字段，展示压力等级指示器
- 底部提供"需要进一步帮助"入口，可引导至学校心理咨询中心联系方式
- 隐私提示：页面顶部固定提示"对话内容严格保密，不做个人身份关联"（对应任务 15）

### 4.3 个性化资源推荐

> NEW v0.2 — 对应任务 13：Coze Agent 在学生画像基础上匹配资源

**页面内容**：
- 基于学生画像（学业状态、挂科情况、年级专业等脱敏特征），由 AI 推荐个性化学习资源
- 卡片列表形式展示推荐内容：
  - 学习资料（课件、复习指南、视频课程）
  - 辅导活动（学习小组、答疑时间、讲座通知）
  - 心理调适资源（减压音频、放松技巧）
- 每条资源卡片：标题 + 摘要 + 类型标签 + 查看详情按钮
- 可标记"已读/已收藏"

### 4.4 我的成长档案

> NEW v0.2 — 对应任务 14：学生成长记录时间轴前端界面

**页面内容**：
- **时间轴组件**: 按学期纵向排列，展示学生的关键节点：
  - 学业成绩变化趋势（GPA 曲线）
  - 挂科/补考记录
  - 帮扶干预记录
  - 心理关怀记录
- **数据可视化**（小型图表）:
  - 学分完成进度环形图
  - 成绩变化折线图
- 学期切换器，查看不同学期的档案
- 数据均为脱敏后的本人可见内容

### 4.5 个人中心

**页面内容**：
- 用户头像（专属形象）+ 基本信息（用户名、学号/工号、院系）
- 账户安全：修改密码入口
- "我的反馈"列表：历史上报的未识别问题及处理状态
- 退出登录

## 5. 管理端功能

### 5.1 数据看板（管理端首页）

**对应 API**：

| 接口 | 路径 |
|------|------|
| 热点提问分析 | `GET /api/stat/hot-questions` |

> 对应任务 9：ECharts 柱状图、折线图可视化；任务 10：重点热门问题主动推送

**页面内容**：

**5.1.1 统计图表区**
- **热点问题词云图**（ECharts）：展示近 N 天高频关键词
- **提问趋势折线图**：按日/周/月展示提问量变化
- **业务分类柱状图**：按业务模块（选课、考务、成绩、心理等）统计提问分布
- **Top 问题排行榜**：排名列表展示

**5.1.2 筛选条件**
- 时间范围选择（7/14/30 天 / 自定义）
- 业务模块筛选
- 展示数量调整（Top 10 / 20 / 50）

**5.1.3 摘要卡片**
- 总提问数、活跃用户数、未处理问题数、今日转人工次数

**5.1.4 主动推送配置区**
- 管理员可设置"重点热门问题"→ 这些会被推送到学生端首页的跑马灯公告或"猜你想问"列表
- 推送方式选择：首页推荐 / 弹窗通知 / 跑马灯公告
- 对应任务 10

### 5.2 知识库管理

**对应 API**：

| 接口 | 路径 |
|------|------|
| 上传并解析文件 | `POST /api/kb/upload` |
| 查询问答词条列表 | `GET /api/kb/entries` |
| 手动新增问答 | `POST /api/kb/entries` |

> 对应任务 7（文件上传支持 Excel/Word）、任务 8（CRUD 管理表格）

**页面内容**：

**5.2.1 知识库词条列表页**
- 表格展示所有 Q&A 词条：问题、答案摘要、分类标签、创建时间、最后修改时间
- 搜索框：按关键词搜索问题和答案
- 分类筛选：按业务模块（选课/考务/成绩/学籍/生活等）过滤
- 操作按钮：编辑（弹窗修改）、删除（二次确认）、置顶/热门标记
- 分页器

**5.2.2 新增词条弹窗/页面**
- 问题输入框
- 答案输入框（富文本编辑器，支持简单排版）
- 分类标签选择（下拉/多选）
- 提交 → `POST /api/kb/entries`

**5.2.3 批量导入**
- 上传区域（拖拽 + 文件选择器）：支持 `.xlsx` 和 `.docx`
  - v0.1 仅提 Excel，v0.2 根据任务 7 增加 Word 支持
- 上传模板下载链接（提供标准 Excel 模板）
- 上传进度条
- 解析结果预览（成功/失败条数、失败原因）
- 确认导入按钮

### 5.3 人工客服台

**对应 API**：

| 接口 | 路径 |
|------|------|
| 客服回复用户 | `POST /api/chat/admin/reply` |
| 结束会话交还 AI | `POST /api/chat/admin/finish` |
| *获取待处理会话列表* | *待后端补充* |

> 对应任务 5：WebSocket 实时通讯 + 管理端人工客服工作台

**页面内容**：

**会话列表（左侧栏）**
- 分 Tab：等待中 / 接待中 / 已结束
- 每个会话项展示：学生匿名标识、问题摘要、等待时长计时器
- 高亮新消息提醒（橙色圆点）
- 客服可"接入"等待中的会话

**对话区（右侧主区域）**
- 选中会话后展示完整历史对话（AI 阶段 + 人工阶段分隔线）
- 底部输入框 + 快捷回复短语（可预设模板："请稍等，我帮您查询..."等）
- 消息基于 WebSocket 实时收发

**操作区**
- "结束会话"按钮 → `POST /api/chat/admin/finish` → 会话移入"已结束"Tab
- 会话备注：客服可为会话添加内部备注（仅管理端可见）
- 当前接待数 / 总排队数 统计

### 5.4 未识别问题管理

**对应 API**：

| 接口 | 路径 |
|------|------|
| 未识别问题列表 | `GET /api/admin/unrecognized/list` |
| 更新问题状态 | `POST /api/admin/unrecognized/update-status` |

> 对应任务 11：未识别问题列表页，供管理员查阅

**页面内容**：
- **表格列表**: 分页展示，列：问题内容、提问时间、提问次数（汇总去重）、当前状态
- **状态筛选**: 全部 / 未处理 / 已处理 / 已补充至知识库

  - "忽略"（针对无意义内容）
- **统计卡片**: 未处理总数、本月新增、本月已处理
- **分页器**

| `/admin/academic/reports` | 帮扶报告与档案 | | | ● | ● | ● |
| `/admin/profile` | 管理端个人中心 | | ● | ● | ● | ● |

### 6.2 鉴权策略

- **路由守卫**: 未登录 → 跳转 `/login`
- **角色守卫**: 根据 `role` 字段限制路由访问，无权限 → 跳转对应首页 + Toast 提示
- **功能级权限**: 同一页面内，按钮/操作根据角色显示/隐藏（如知识库"删除"按钮仅管理员可见）
- **Token 管理**: localStorage 存储，Axios 拦截器自动注入 `Authorization` Header
- **Token 过期**: 401 响应 → 清除 Token → 跳转登录页

### 7.2 关键交互流程

#### 7.2.1 转人工闭环流程

```
学生发送问题 → AI 无法识别 → 学生点击"转人工"
  → POST /api/chat/transfer-to-human → 返回排队号
  → 前端轮询/WebSocket 等待状态变更
  → 客服接入 → WebSocket 建立 → 双向对话
  → 客服结束 → POST /api/chat/admin/finish → 学生端切回 AI 模式
```

#### 7.2.2 未识别问题 → 知识库闭环

```
用户提问 → Coze 工作流未匹配 → 自动记录到未识别问题表
  → 管理端查看未识别问题列表
  → 管理员"添加到知识库" → 预填问题 → 编辑答案 → 入库
  → 下次用户提问 → Coze 工作流命中知识库 → 返回标准答案
```

#### 7.2.3 学业帮扶闭环

```
辅导员选择学生 + 学期 → 生成预警
  → Coze 工作流分析学业数据 → 输出帮扶策略
  → 辅导员审阅 → 保存为帮扶记录
  → 实施帮扶措施 → 后续学期对比数据
  → 生成成长档案 + 成效报告 PDF
```
